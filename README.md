# Devpulse

Devpulse is a minimal developer management system built with **Spring Boot** and **PostgreSQL**.  
It allows creating, updating, deleting, and searching developers with skills and email information.

## Features

- Create new developers
- Update developer information
- Delete developers
- Search developers by name or skill
- Pagination and sorting
- User authentication using Spring Security
- Role-based access (ADMIN, USER)

## Getting Started

1. Clone the repository
2. Configure PostgreSQL and update `application.properties`
3. Run the Spring Boot application
4. Sign in using the default credentials:
    - **Username:** admin
